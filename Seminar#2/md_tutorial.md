## как добавить списки

чтобы создать список с помощью разметки MarkDown необходимо прописать следующию клнструкцию:

со звёздочкой "*" для не нумеровонных списков:

```
* элемент 1
* Элемент 2
* элемент 3
    * элемент 3.1
```
Пример:
* элемент 1
* Элемент 2
* элемент 3
    * элемент 3.1

с цифрой и точкой перед элементом для нумеровонного списка:

```
1. элемент 1
2. элемент 2
1. элемент 3
    0. элемент 3.1
```
Пример:
1. элемент 1
2. элемент 2
1. элемент 3
0. элемент 4
