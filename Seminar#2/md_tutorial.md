## как добавить исходный код

Чтобы добавить исходный код,
необходимо использовать конструкцию следующего вида:
```html
<table>
    <tr>
        <td style="color:00B74A">
            text
        </td>
    </tr>
</table>
```
Пример рабочего кода без ковычек:
<table>
    <tr>
        <td style="color:00B74A">
            text
        </td>
    </tr>
</table>
